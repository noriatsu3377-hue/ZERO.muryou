肩甲上腕リズム ― 腕を挙げる"2:1の共同作業"

― 肩関節だけでは腕は挙がらない。肩甲骨との協調が鍵 ―

はじめに

肩甲骨面編で「腕を挙げるとき、肩甲骨と上腕骨は約2:1で協調する」と触れた。今回はその肩甲上腕リズム（Scapulohumeral Rhythm）を主役に、腕がどう挙がるのかを掘り下げる。

肩は「肩関節だけ」で挙がっているのではない。肩甲骨と上腕骨の絶妙な共同作業で、あの大きな可動域が生まれる。このリズムが崩れると肩の不調が起こる。トレーナー・セラピスト向けに無料版の範囲で整理する。

1. 肩甲上腕リズムとは ― 2:1の比率

腕を横から頭上まで（約180度）挙げるとき、肩甲上腕関節で約120度、肩甲骨の上方回旋で約60度が分担される。この2:1の協調が肩甲上腕リズム。

〈運動学的視点〉 もし肩関節だけで挙げようとすると、可動域は限られ、途中で骨がぶつかる。肩甲骨が上方回旋して関節窩を上に向け続けることで、上腕骨が最後まで滑らかに挙上できる。役割分担で大可動域を実現する仕組みだ。

2. 挙上の各相 ― 肩甲骨はいつ動くか

挙上は一様ではなく、相によって主役が変わる。

  • 初期（〜約30度） … 主に肩関節が動く。肩甲骨は胸郭上で位置を定める"セッティング相"（個人差あり）
  • 中期〜後期 … 肩甲骨の上方回旋が本格化し、2:1に近い比率で協調する
  • 終盤 … 鎖骨・胸鎖関節・肩鎖関節も連動し、肩甲骨がさらに回旋・後傾する

つまり肩甲骨は「肩の土台を動かし続けるパートナー」。前鋸筋・僧帽筋のフォースカップル（前鋸筋編・フォースカップル編）が、この上方回旋を生んでいる。

3. リズムを支える要素

  • 肩甲骨の上方回旋フォースカップル … 僧帽筋上部・下部＋前鋸筋下部の協調
  • ローテーターカフ … 骨頭を関節窩に安定させ、三角筋とのフォースカップルで滑らかな挙上を保つ
  • 肩甲骨面（スキャプラプレーン） … 適切な面の上でこそリズムが成立（肩甲骨面編）
  • 胸椎・胸郭の可動性 … 土台が動くことで肩甲骨が回旋できる（Joint by Joint編・横隔膜編）

4. リズムが崩れると ― スキャプラディスキネジア

肩甲骨の動きの異常をスキャプラディスキネジアと呼ぶ。2:1の協調が崩れると、次のような問題が起こる。

  • 肩インピンジメント … 上方回旋が不足し、挙上時に肩峰下スペースが狭くなる（肩甲骨面編）
  • 翼状肩甲・肩甲骨の浮き … 前鋸筋の弱化で肩甲骨が胸郭から離れる（前鋸筋編）
  • 僧帽筋上部の過活動 … 肩をすくめて代償し、肩こり・首の緊張へ
  • 可動域制限・挙上時痛 … 「腕を上げると痛い・引っかかる」
  • 腱板への負担増 … リズムの乱れで骨頭が不安定になり腱板ストレスが増える

5. まず押さえたい基本の考え方（基礎編）

  • 肩は"肩甲骨と上腕骨の共同作業" … 肩関節だけを見ない
  • 挙上を後方から観察 … 肩甲骨が滑らかに上方回旋しているか、すくみ代償がないか
  • 土台（胸椎・胸郭）から見る … 土台が硬いとリズムは崩れる
  • すくみ（僧帽筋上部優位）に注意 … 上方回旋の不足を肩挙上で代償していないか

6. 「構造・神経・流れ」で立体的に見る

肩甲上腕リズムは構造（肩甲骨・上腕骨・鎖骨の配置）に見えて、その協調を生むのはフォースカップルという神経制御であり、胸郭・呼吸という流れにも支えられる。

だから「肩が挙がらない→肩だけ動かす」では戻りにくい。肩甲骨の回旋・協調・土台を三層で読む――これが零 -ZERO- の統合的な見方だ。

おわりに

腕が頭上まで滑らかに挙がるのは、肩甲骨と上腕骨が2:1で協調する肩甲上腕リズムのおかげだ。肩は単独の関節ではなく、土台（肩甲骨）とアーム（上腕骨）の共同作業。このリズムが崩れると、インピンジメントや肩こりが生まれる。

肩を"点"でなく"協調するシステム"として捉える。それは、身体を部分でなく全体で読み、本来へ戻すZEROの考え方そのものだ。
