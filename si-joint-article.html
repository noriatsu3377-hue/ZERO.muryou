仙腸関節の理解 ― 動かないのに動く"荷重の要"

― 上半身と下半身の力が乗り換える、骨盤のクロスポイント ―

はじめに

仙骨編で仙骨の前傾（ニューテーション）を、胸腰筋膜編で後斜スリングによる仙腸関節のロックを扱った。今回はその要である仙腸関節（Sacroiliac Joint：SIJ）そのものを主役にする。

仙腸関節は「ほとんど動かない」のに「わずかに動く」、そして上半身と下半身の荷重が乗り換える決定的なポイントだ。慢性腰痛の一部は、実は腰椎ではなくこの関節に由来する。トレーナー・セラピストが押さえておきたい基礎を、無料版の範囲で整理する。

1. 仙腸関節とは ― 半関節という特殊な存在

仙腸関節は、仙骨と左右の寛骨（腸骨）をつなぐ関節。骨盤の後方にある。

可動域が大きい四肢の関節（滑膜関節）と、ほぼ動かない縫合の中間。強靭な靭帯で固められ、動きはごくわずか（数mm・数度）の「半関節（アンフィアルトローシス）」だ。

関節面は平面ではなく凹凸が噛み合う不整な形をしており、加齢とともに面が粗くなり、噛み合わせ（摩擦）で安定性が増す。つまり仙腸関節は「大きく動くための関節」ではなく「安定して力を伝えるための関節」だ。

2. わずかに動く ― ニューテーションとカウンター

動きは小さいが、ゼロではない。仙骨が寛骨に対してわずかに前後に傾く（仙骨編で扱った動き）。

  • ニューテーション（うなずき） … 仙骨底が前下方へ。関節が締まり、安定する。荷重位・立位で優位
  • カウンターニューテーション（起き上がり） … 仙骨底が後上方へ。関節が緩む。臥位・非荷重で優位

この「立って支えるときは締まり、休むときは緩む」微小な切り替えが滑らかに起こることが、仙腸関節の健全さの条件だ。

3. 自己ロックの2機構 ― フォーム&フォースクロージャー

仙腸関節が「わずかしか動かないのに強く安定する」秘密が、2つのクロージャー（締結）機構だ。

① フォームクロージャー（形状による安定）

関節面の形状・凹凸の噛み合わせによる受動的な安定。骨の形そのものが生む安定で、いわば"くさび"がはまる仕組み。仙骨は上から押されるほど寛骨の間にくさび状に食い込み、安定が増す。

② フォースクロージャー（筋・筋膜の張力による安定）

筋・筋膜・靭帯の張力による能動的な安定。胸腰筋膜編で扱った後斜スリング（広背筋⇄大殿筋）、大殿筋・多裂筋・骨盤底筋・胸腰筋膜が張ることで、関節面を圧着させる。動作に応じて安定度を調整できるのが特徴だ。

形（フォーム）と力（フォース）――この2つが揃って、仙腸関節は"動かず、しかし柔軟に"荷重を支える。

4. 役割 ― 力が"乗り換える"クロスポイント

仙腸関節の最重要の役割は荷重伝達だ。

  • 上半身の重み … 脊柱 → 仙骨 → 仙腸関節 → 寛骨 → 股関節 → 下肢へと下る
  • 地面からの反力 … 下肢 → 寛骨 → 仙腸関節 → 仙骨 → 脊柱へと上る

つまり仙腸関節は、体幹と下肢の力が互いに乗り換える"クロスポイント"。歩行では左右交互にニューテーション/カウンターが起こり、後斜スリングと連動して推進力を伝える。運動連鎖編の「上行性・下行性が交わる場所」でもある。

5. 仙腸関節が機能しないと起こること

  • 仙腸関節性の腰殿部痛 … PSIS周辺〜殿部の痛み。腰椎由来と紛らわしいが起点が異なる
  • 不安定（ゆるみすぎ） … フォースクロージャー不全（殿筋・スリングの弱化）で関節がぐらつく。産後などで起こりやすい
  • 固着（動かなさすぎ） … 切り替えが失われ、隣接の腰椎・股関節が代償する
  • 荷重伝達の破綻 … 力の乗り換えが乱れ、片側荷重・歩行効率の低下
  • 連鎖的な波及 … 上は腰椎、下は股関節・膝へ症状が飛ぶ（運動連鎖編）

「腰の下のほう・お尻が痛い」「片脚立ちや寝返りでズキッとする」といった訴えは、仙腸関節を疑う手がかりになる。

6. まず押さえたい基本の考え方（基礎編）

無料版では、現場で意識したい"原則"だけを示す。

  • 動かすより"締める"関節 … 仙腸関節は可動より安定が本分。ゆるみには安定化を
  • フォースクロージャーを働かせる … 大殿筋・多裂筋・後斜スリングの働きが関節を締める鍵
  • 腰椎痛と切り分ける … 痛みの位置（PSIS周辺）や動作で、腰椎由来かSIJ由来かを意識する
  • 不安定か固着かを見極める … ゆるみすぎと固まりすぎでは対応が真逆になる
ここから先――仙腸関節の疼痛誘発テスト（クラスターテスト）、不安定/固着の鑑別、フォースクロージャーを高める殿筋・スリングの賦活ドリル、腰椎・股関節との切り分けと症例アプローチは、有料コミュニティ「零 -ZERO- MEMBERS」で体系的に扱います。無料版は"理解"まで、有料版は"現場で結果を出す実践"まで、が役割分担です。

7. 「構造・神経・流れ」で立体的に見る

仙腸関節は構造（フォームクロージャー＝骨形状）で受動的に、神経（フォースクロージャー＝筋の協調・運動制御）で能動的に安定し、荷重・呼吸という流れ（横隔膜〜骨盤底の圧）にも支えられる。

だから「お尻が痛い→そこを揉む」では戻りにくい。形の安定・筋の締結・荷重の乗り換えを三層で読む――これが零 -ZERO- の統合的な見方だ。

おわりに

仙腸関節は、ほとんど動かないのにわずかに動き、形と力の2つで自己ロックしながら、上半身と下半身の荷重を乗り換えさせる"要"だ。可動域の大きさではなく、安定と伝達の質で評価すべき関節である。

揺るがす対象ではなく、締まるべきときに締まり、緩むべきときに緩む本来の働きへ戻す。それは、身体を「足す前に戻す」という零の考え方そのものだ。
